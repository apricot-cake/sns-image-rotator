# SNS Image Rotator（SNS画像回転）

SNS のタイムラインに流れてくる横倒しの画像を、その場で正しい向きに回転できる Chrome 拡張機能。X (Twitter) と Bluesky (bsky.app) に対応（Bluesky はタイムライン写真の回転。拡大表示〈ライトボックス〉での回転は今後対応予定）。

![横倒しのイラストを右クリックメニューから 90° 回転させ、正しい向きで大きく表示するまで](docs/screenshot.png)

UI は多言語対応（`_locales`, en / ja / ko / zh_CN / zh_TW）: 右クリックメニュー・ホバーボタンのツールチップ・ポップアップの文言と、拡張機能の表示名を、ブラウザの言語に合わせて出し分ける。表示名は "SNS" が通じる地域（日本・韓国）では「SNS画像回転」「SNS 이미지 회전」、通じない地域では英語 "Social Media Image Rotator" や中文「社交媒体图片旋转器」を使う。

## なぜ既製の回転系拡張機能では駄目なのか

X はツイート画像を `<img>` ではなく `<div>` の CSS 背景画像として描画し、`<img>` は透明な当たり判定として重ねている。汎用の回転拡張機能は `<img>` を回すため、X 上では見た目が変わらない。本拡張は背景 `<div>` と `<img>` の両方を回すことでこれを解決する。

## 機能

- **ホバーボタン**: 画像にマウスを乗せると右上（拡大表示では右下）に左回転・右回転の 2 ボタンが出る。クリックごとに 90° 回転する。ポップアップ設定でオンオフ可（既定オフ）
- **右クリックメニュー**: 「画像を左/右に 90° 回転」。常に有効
- **拡大表示では画面いっぱいに拡大**: ライトボックスでは、回転後の画像を元の枠ではなくモーダルの使える領域に合わせて拡大する。横向き投稿を縦に回すと、画面の高さをフルに使って大きく表示される
- **タイムラインは枠ごと可変**: 単独写真の投稿は、回転時に枠自体を回転後の画像の縦横比に作り替え、ツイート本文と同じコンテンツ列の幅いっぱいに（余白なく）大きく表示する。縦長ファイルの横向き絵など、元は列幅より狭く表示されていた画像も回転後は列幅まで拡大される。ただし横長→縦長で縦に長くなりすぎる場合は、画面（ビューポート）の高さを超えないよう高さを抑え、画像全体が一度に見えるようにする。このとき枠は画像にぴったり合わせて（灰色の帯を出さず）列の中央に置く。複数枚グリッドと動画サムネイルは、段組みやプレイヤーを崩さないよう従来どおり元の枠内に縮小する
- **対象**: タイムライン・引用ポストの写真、および拡大表示（ライトボックス）

## ストア説明文

**SNSで流れてくる、あの横倒しのイラスト。**

スマホなら端末をくるっと傾ければ見られるけど、デスクトップだと首をかしげるしかない——そんな経験、ありますよね。

この拡張機能は、画像を右クリックして「画像を90°回転」を選ぶだけ。**その場で、絵をいちばん大きく見える向きに直します。** よく回すなら、設定でホバー回転ボタン（マウスを乗せると出るワンクリックのボタン）もオンにできます。

- 左右どちらにも 90° 回せる
- 回した絵は見やすいサイズにぴったり収まる
- 対応: X（Twitter）／ Bluesky

## インストール

[Chrome ウェブストア](https://chromewebstore.google.com/detail/affodfbfgaikjohkhnfmjalnapgjfklf) から入れられる。

公開直後の拡張機能には Chrome の「セーフ ブラウジング保護強化機能」が信頼済みと判定するまで数か月かかるため、その間はインストール時に警告が出る。危険と判定されたわけではなく、実績を積むまでの既定の挙動。

## インストール（開発版）

1. `npm install && npm run build`
2. Chrome の `chrome://extensions` で「デベロッパー モード」をオン
3. 「パッケージ化されていない拡張機能を読み込む」で `.output/chrome-mv3` を選択

コード更新後は同じ画面で拡張機能の再読み込みボタン（↻）を押す。

## 開発

[WXT](https://wxt.dev/) + TypeScript。

```sh
npm install
npm run dev      # 自動リロード付き開発
npm run build    # 本番ビルド
npm run compile  # 型チェック
```
